# Host-Cloud.org 游戏平台

一个简单的HTML5游戏平台网站，集成了多个经典小游戏。

## 包含游戏

- **2048游戏**：经典数字合成游戏
- **贪吃蛇**：控制蛇吃食物并避免碰撞
- **植物大战僵尸**：经典策略防御游戏

## 技术栈

- 前端：纯HTML/CSS/JavaScript
- 后端：Node.js + Express

## 使用方法

1. 克隆仓库
```bash
git clone https://github.com/yourusername/host-cloud.org.git
cd host-cloud.org
```

2. 安装依赖
```bash
npm install
```

3. 启动服务器
```bash
node server.js
```

4. 在浏览器中访问 `http://localhost:3000`

## 特性

- 简洁现代的界面设计
- 响应式布局
- 游戏内嵌iframe加载
- 游戏说明及操作指引
